# CSS 
Es un lenguaje de diseño utilizado para definir el aspecto visual y el formato de nuestra aplicacion, esta permite separar el contenido estructural (HTML) de la presentación visual, lo cual facilita la gestión y mantenimiento del sitio web, es esencial para mejorar la presentación y la experiencia de usuario en las páginas web, proporcionando control detallado sobre cómo se visualizan.
